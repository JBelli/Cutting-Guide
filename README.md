# Cutting-Guide
Human postmortem brain container: stereotaxic enclosure &amp; integral cutting guide. Constructed of Victrex AM™ 200; high mechanical strength &amp; chemical resistance. Brains embedded in agar/sucrose &amp; container scanned at 7T MRI. The cutting guide facilitates coronal sectioning. Block face photographs are co-registered with MRI for WMH tissue sampling.
